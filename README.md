# Intellectual Floor  
### A Quantitative Idea Ranking Tool for Solo Builders

When you have five ideas and only one weekend, how do you choose?

Most people pick based on mood, excitement, or recency bias.

**Intellectual Floor** replaces guesswork with structured evaluation.

It lets solo builders define what “good” means, score ideas against those criteria, automatically rank them, and then generate stronger alternatives using a data-informed threshold.

Built entirely client-side. No backend. No dependencies. Fully offline.

---

## 🚀 What It Does

1. Define 7 custom scoring criteria (1–10 scale)
2. Score each idea across those criteria
3. Automatically calculate weighted or simple averages
4. Rank ideas in real time
5. Compute an **Intellectual Floor**
6. Generate an AI upgrade prompt to create better ideas

Everything runs in the browser using localStorage.

---

## 🧠 The Core Concept: The Intellectual Floor

Instead of asking:

> “Which idea do I feel like building?”

You ask:

> “Which idea objectively clears my quality threshold?”

The app:

- Identifies your top-scoring idea
- Sets a floor (Top Score − 0.5 by default)
- Visually flags ideas below that threshold
- Forces new ideas to beat your current best

It turns ideation into an optimization loop.

## ⚙ Features

### Custom Criteria
- Add / remove criteria
- Rename and define scoring rules
- Toggle reverse scoring (e.g. complexity)
- Assign weights in weighted mode

### Flexible Scoring
- Simple average
- Weighted average
- Reverse scoring supported
- Live recalculation

### Idea Management
- Add unlimited ideas
- Score with sliders
- Duplicate / delete ideas
- Add notes to increase score clarity

### Intellectual Floor Modes
- Top − 0.5
- Top − 1.0
- 90th percentile

### AI Upgrade Prompt
Generates a ready-to-copy structured prompt to feed into ChatGPT to create ideas that score above your current floor.
